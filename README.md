# picohttp

picohttp is an extremely simple, and lightweight HTTP server written in C.

## Installation

Clone the repo:

```bash
git clone https://github.com/bigkahuna00/picohttp
```

Use the makefile:

```bash
make
```
Or compile using gcc:
```bash
gcc picohttp.c -o picohttp
```

## Usage

```bash
./picohttp <PORT> <HTML FILE>
```
